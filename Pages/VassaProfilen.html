Vässa profilen


Många känner att det är svårt att framhäva sig själv, hur gör man för att det inte ska bli för skrytsamt eller för blygsamt? Det är viktigt att ge en så sann bild av dig själv som möjligt för att öka chanserna att träffa någon som verkligen matchar. Likaså är det avgörande vilka bilder du väljer, detta kan vi hjälpa dig med. Jeanette och Anna på J&A Matchmaking arbetar som professionella matchmakers, de kan hjälpa dig att vässa din profil.

Detta kan vi hjälpa dig med:
  • Att välja den profilbild som framhäver dig på bästa sätt.
  • Att välja bra bilder av dig till ditt album.
  • Vi hjälper dig formulera din presentation så den blir tilltalande och framhäver dig på bästa sätt.
  • Vi kan hjälpa dig formulera din text så att du uppmärksammas av den sortens match som du söker.
